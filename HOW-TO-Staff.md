# How to Use the System - Staff Guide

## Overview
As **Staff**, you have limited access to view system information. Your primary function is to assist with event operations and may have access to view certain information depending on your specific assignment.

## Getting Started

### Logging In
1. Navigate to the login page
2. Enter your username and password
3. Click "Login"
4. You'll be redirected based on your assignment

### Dashboard Overview
Staff access varies based on how your account is configured. You may have access to:
- View classes and schedules
- View user information (if permitted)
- Access to check-in functions (if assigned)
- Limited reporting capabilities (if assigned)

## Profile Management

### Editing Your Profile
1. Click on your name in the top banner (if visible and clickable)
2. You can update:
   - First Name and Last Name
   - Email (optional for Staff)
   - Phone
   - Investiture Level
   - Password (local users only)
3. Click **"Update Profile"**

**Note**: 
- For Stytch users (external authentication), password changes must be done through Stytch's password reset feature on the login page
- Changing email for Stytch users sends a verification email to the new address
- You cannot change your club assignment or role

## Common Tasks

### Viewing Information
- Staff may have view-only access to:
  - Class schedules
  - User rosters
  - Event information
  - Check-in status

### Checking In Participants (If Assigned)
1. Navigate to Check-In section (if available)
2. Search for participants by name
3. Mark participants as checked in
4. Verify check-in numbers if displayed

## Best Practices

1. **Follow Instructions**: 
   - Staff roles are typically configured for specific tasks
   - Follow guidelines provided by Event Admins or Administrators
   - Ask for clarification if unsure about your permissions

2. **Data Accuracy**:
   - Verify information before making changes
   - Double-check participant names
   - Report any discrepancies

3. **Communication**:
   - Contact Club Directors for club-specific questions
   - Contact Event Admins for event-related questions
   - Contact Administrators for system access issues

## Troubleshooting

**Can't access features I need:**
- Staff access is limited based on assignment
- Contact an Event Admin or Administrator for additional permissions
- Verify your role is correctly assigned

**Can't log in:**
- Verify your username and password
- Check if your account is active (contact an Administrator)
- Use password reset if needed (for Stytch users, use "Forgot Password" link)

**Seeing unexpected information:**
- Contact an Administrator to verify your access level
- Some features may be restricted for Staff role

## Support

For issues or questions:
1. Review this guide
2. Check system notifications for error messages
3. Contact your **Event Admin** for:
   - Event-specific questions
   - Feature access requests
   - Operational guidance
4. Contact an **Administrator** for:
   - System access issues
   - Role and permission questions
   - Technical problems
