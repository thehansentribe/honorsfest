# How to Use the System - Club Director Guide

## Overview
As a **Club Director**, you manage your club's participants and classes for the events your club is assigned to. You can add users, create classes, generate registration codes, check in participants, and view reports for your club.

## Getting Started

### Logging In
1. Navigate to the login page
2. Enter your username and password
3. Click "Login"
4. You'll be redirected to the Club Director Dashboard

### Dashboard Overview
The Club Director Dashboard has the following tabs:
- **Users** - Manage users in your club
- **Classes** - View and create classes for your club
- **Registration Codes** - Generate codes for students to register themselves
- **Check-In** - Check in your club's participants at events
- **Reports** - Generate reports for your club

**Note**: If your club is assigned to multiple events, you'll see an event selector dropdown in the top banner. Select an event to manage data for that specific event.

## Tab-by-Tab Guide

### Users Tab

#### Creating a New User
1. Click the **"Add User"** button
2. Fill in required fields:
   - **First Name** and **Last Name**: Required
   - **Date of Birth**: Required
   - **Email**: Optional for Teachers, Students, and Staff
   - **Phone**: Optional
   - **Role**: Select from Teacher, Student, or Staff
   - **Club**: Automatically set to your club (cannot be changed)
   - **Event**: Automatically set based on your club's event assignments
   - **Investiture Level**: Select from available options
   - **Password**: Leave blank to use default "password123" (local users only)
3. Click **"Create User"**

#### Bulk Adding Users
1. Click **"Bulk Add Users"** button
2. Fill in the grid with multiple users at once:
   - Each row represents one user
   - Fill in First Name, Last Name, Date of Birth, Email, Phone, Role, Investiture Level
   - Leave rows blank to skip them
3. Click **"Create All Users"** to create all filled-in rows at once

**Bulk Add Tips**:
- Copy/paste from Excel or Google Sheets
- Dates should be in YYYY-MM-DD format
- Roles: Teacher, Student, or Staff only
- All users will be assigned to your club automatically

#### Editing a User
1. Find the user in the users list
2. Click **"Edit"** button
3. Make changes:
   - You can update user details
   - For **Local users**: You'll see "(Local)" and can change password directly
   - For **Stytch users**: Password changes must be done through Stytch's password reset
   - Changing email for Stytch users sends a verification email
4. Click **"Update User"**

#### Deactivating a User
1. Find the user in the list
2. Click **"Deactivate"** button
3. Deactivated users are removed from all classes and cannot log in

### Classes Tab

#### Viewing Classes
- See all classes created by or for your club
- Filter by event (if your club is in multiple events)
- View active and inactive classes
- See class details: Honor, Teacher, Location, Timeslot, Capacity, Remaining Spots

#### Creating a Class
1. Select an **Event** (if you have multiple events)
2. Click **"Create Class"** button
3. Fill in required fields:
   - **Honor**: Select from the list of available honors
   - **Teacher**: Optional - can be assigned later (must be a teacher from your club)
   - **Location**: Will be assigned by Admin later (you can proceed without it)
   - **Timeslots**: Select one or more timeslots when this class will be held
   - **Max Capacity**: Maximum number of students
4. Click **"Create Class"**

**Note**: 
- Classes you create are associated with your club
- Locations are typically assigned by Admins after class creation
- Teachers can be assigned later if needed

#### Editing a Class
1. Find the class in the list
2. Click **"Edit"** button
3. Update class details, teacher, or capacity
4. Click **"Update Class"**

**Note**: Only classes created by or associated with your club can be edited.

### Registration Codes Tab

#### Generating Registration Codes
1. Select an **Event** (if you have multiple events)
2. Click **"Generate Registration Code"** button
3. Set expiration days (default: 30 days)
4. Click **"Generate Code"**
5. A unique registration code is created and displayed

#### Using Registration Codes
- Share the registration code with students who need to register
- Students visit the registration page and enter the code
- Codes expire after the set number of days
- Each code can only be used once
- Codes are specific to your club and the selected event

**Best Practice**: Generate codes for groups of students and track which codes have been used.

#### Viewing Registration Codes
- See all codes you've generated
- View expiration dates
- See which codes have been used
- Codes show associated club and event

### Check-In Tab

#### Checking In Participants
1. Select an **Event** (if you have multiple events)
2. The list shows all students from your club for that event
3. Click **"Check In"** next to a participant's name
4. Check-in status is updated in real-time

#### Confirm Attendance Feature
1. After checking in participants, click **"Confirm Attendance & Remove Unchecked-In Students from Your Club"**
2. This action will:
   - Remove all unchecked-in students from your club from their registered classes
   - Update waitlists for affected classes
   - **Important**: This ONLY affects members of YOUR club
   - Students from other clubs are not affected
3. Confirm the action when prompted

**Warning**: 
- Unchecked-in students will need to re-register for classes after being checked in
- This action cannot be undone
- Only affects your club's members

**Use Case**: Use this after registration/check-in closes to finalize your club's attendance and free up spots for waitlisted students from other clubs.

### Reports Tab

#### Generating Reports
1. Select an **Event** (if you have multiple events)
2. Click **"Generate Report"** or **"Export CSV"**
3. Reports include:
   - Student Name
   - Club (your club)
   - Honor Name (Class)
   - Timeslot
   - Attended Status
   - Completed Status
   - Investiture Level

**Note**: Reports only show data for YOUR club, not all clubs in the event.

Reports can be exported as CSV files for further analysis in Excel or Google Sheets.

## Event Selection

### Multiple Events
If your club is assigned to multiple events:
- Use the **Event Selector** dropdown in the top banner
- Each event is managed separately
- Classes, registration codes, and reports are event-specific
- Check-in is event-specific

**Tip**: The event selector remembers your last selection.

## Profile Management

### Editing Your Own Profile
1. Click on your name in the top banner (it's underlined and clickable)
2. You can update:
   - First Name and Last Name
   - Email (required for Club Directors; for Stytch users, this sends a verification email)
   - Phone
   - Investiture Level
   - Password (local users only)
3. Click **"Update Profile"**

**Note**: You cannot change your club assignment or role through your profile.

## Tips and Best Practices

1. **User Management**:
   - Use bulk add for importing lists of students
   - Keep user information up to date
   - Deactivate users who are no longer participating

2. **Class Creation**:
   - Coordinate with event Admins about locations
   - Assign teachers when available
   - Set realistic capacity limits
   - Create classes early to allow time for registration

3. **Registration Codes**:
   - Generate codes before registration opens
   - Track which codes you've distributed
   - Set appropriate expiration dates
   - Generate new codes if previous ones expire

4. **Check-In Process**:
   - Check in participants as they arrive at the event
   - Use "Confirm Attendance" after check-in closes to finalize attendance
   - This helps free up spots for waitlisted students

5. **Multi-Event Management**:
   - Switch between events using the event selector
   - Each event maintains separate classes and registrations
   - Generate event-specific registration codes

6. **Communication**:
   - Share registration codes securely
   - Inform students about registration deadlines
   - Update students about class changes

## Troubleshooting

**Can't see my club:**
- Verify you're assigned as a Club Director (contact Admin)
- Check your login credentials
- Refresh the page

**No events showing:**
- Your club may not be assigned to any events yet
- Contact an Admin to link your club to events
- You'll see a banner if no events are assigned

**Registration codes not working:**
- Check if code has expired
- Verify code hasn't already been used
- Ensure code is for the correct event
- Generate a new code if needed

**Can't create classes:**
- Ensure an event is selected
- Verify your club is linked to an event
- Check that timeslots exist for the event

**Students can't register:**
- Verify event status is "Live" (contact Admin)
- Ensure students have registration codes
- Check that classes are Active
- Verify students are in your club
